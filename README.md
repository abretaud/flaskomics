# FlaskOmics


FlaskOmics is the future of [AskOmics](https://github.com/askomics/askomics).

## Requirements

Python3, Python3 virtual env and npm

```bash
sudo apt install python3 python3-venv npm
```

## Install
```bash
./install.sh
```

## Run

```bash
./run.sh
```

FlaskOmics is available at http://localhost:5000
